# node-db-connect

Connection modules for different databases using Node.js

Create a `.env` file with the following info to test the connections:

```text
MONGO_URL=

MYSQL_HOST=
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_DATABASE=

SQLSRV_HOST=
SQLSRV_USER=
SQLSRV_PASSWORD=
SQLSRV_DATABASE=

PG_HOST=
PG_USER=
PG_PASSWORD=
PG_DATABASE=
```
