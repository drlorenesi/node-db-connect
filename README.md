# node-db-connect

Database Connection Modules for Node.js

Create a .env file with the following info to test the connections:

```text
MSSQLUSER=
MSSQLHOST=
MSSQLPASSWORD=
MSSQLDATABASE=

MYSQLUSER=
MYSQLHOST=
MYSQLPASSWORD=
MYSQLDATABASE=

PGUSER=
PGHOST=
PGPASSWORD=
PGDATABASE=
PGPORT=5432
```

NOTE: If you are unable to connect to SQL Server locally, follow the steps detailed in this tutotial by Ashish Sharma:

https://medium.com/zero-equals-false/how-to-connect-node-js-app-with-sql-server-18a176afae46
